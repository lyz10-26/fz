New Unity Project

场景三：
场景动画
动画播放：点击该按钮可以播放整个场景动画。

相机操作
机位显示：点击该按钮可以显示场景中所有摄像机的位置。
自由相机：点击该按钮后，通过按住鼠标左键，可以对场景进行360度全景预览。按住鼠标中间的滚轮，可以对场景整体放大或者缩小。
固定相机：点击该按钮可以以一个固定视角对整个动场景进行预览。（近景）
动画相机： 点击该按钮可以显示特定角度的场景，（俯视）（全景）

灯光操作
灯光显示：点击该按钮可以为动画场景增加光照效果，聚光灯效果，不同方向3盏灯。
灯光1|2|3开关：点击该按钮可以关闭场景中的部分灯光。分别按不同的按钮可以使场景中的灯光依次全部关闭。
